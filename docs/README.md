
## Simple React Chat with Firestore Database from Firebase
Demo: https://cra-ts-chat.vercel.app/chat

Чат работает в режиме реального времени, основан на базе данных от Firebase, имеется темная и светлая тема. 

Stack: 
- Typescript
- React + Hooks
- React-router
- React-hooks-form
- Firebase Firestore
- SCSS
- Animate.css
